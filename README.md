My changes to GatherMate2 only include German and English translations. There are no issues in TWW, but in Midnight, the nodes and farmbar won't work with other languages ​​yet. You're welcome to submit additional languages; search in \locales\GatherMate2-enUS.lua for "-- Midnight Expansion Nodes" and "-- Farmbar Options" and replace "true" with the translation in your language, remembering the quotation marks.
